# QR Scanner Client

Десктопное приложение с GUI для считывания QR-кодов через COM-порт и отправки данных на HTTP-сервер.

## Возможности

- Выбор COM-порта из списка доступных устройств
- Указание IP-адреса сервера
- Считывание данных со сканера (9600 baud, UTF-8)
- Отправка кода на сервер: `POST http://<IP>:8000/api/endpoint`
- Отображение статуса в реальном времени

## Требования
- Python 3.8+
- Сканер QR-кодов, переведённый в режим виртуального COM-порта (VCOM)

## Установка и запуск

```bash
# 1. Создайте виртуальное окружение
python -m venv venv

# 2. Активируйте окружение
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите приложение
python main.py
```

## Сборка в исполняемый файл

```bash
pyinstaller --onefile main.py
```

Результат: `dist/main` (macOS/Linux) или `dist/main.exe` (Windows).

## Использование

1. Переведите сканер в режим VCOM **до** запуска программы
2. Укажите IP-адрес сервера в поле "IP сервера"
3. Выберите COM-порт из выпадающего списка (или нажмите "Обновить порты")
4. Нажмите "Старт сканирования"
5. Статус отправки отображается в нижней части окна
